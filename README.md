# Godisk - 轻量级个人私有云盘系统
**Godisk** 是一款基于 **Go 语言** 开发的极简、高效的个人私有云盘系统。项目设计初衷是**简单部署**与**超低资源占用**，非常适合运行在垃圾 VPS 或家用 NAS 上。

## ✨ 核心特性

* 🚀 **极低占用**：实测运行内存仅需几十 MB，对硬件要求极低。
* 📦 **多分片上传**：支持大文件切片上传，确保弱网环境下的上传稳定性。
* 🌐 **在线预览**：内置视频播放器与图片查看器，支持流媒体播放。
* 📥 **离线下载**：支持 HTTP/HTTPS 链接异步离线下载。
* 📝 **简易便签**：内置便签系统，随时记录文字碎片。
* 🔒 **单用户管理**：专为个人设计，配置简单，安全可控。

## ⚙️ 配置说明

你可以在配置文件中修改以下参数以适配你的环境：

| 参数名称 | 默认值 | 单位 | 说明 |
| --- | --- | --- | --- |
| `port` | `80` | - | 服务监听端口 |
| `workdir` | `pwd` | - | 文件存储的工作目录，默认为程序所在目录 |
| `password` | `admin` | - | 系统登录密码 |
| `max_share_size` | `100` | MB | 单个分享文件的最大限制 |
| `max_upload_size` | `50` | GB | 单次上传任务的最大限制 |
| `deadline_store` | `5` | GB | 存储的剩余空间阈值限制 |
| `sqlite_name` | `app.sqlite` | - | 系统数据库文件名 |

## 🛠️ 快速开始

### 1. 编译 (Go 1.20+)

```bash
go build -o godisk main.go

```

### 2. 运行

```bash
./godisk

```

启动后访问 `http://你的IP:80`，默认密码为 `admin`。
